# 答辩讲稿提纲

## 1. 项目介绍

本项目是 SpringCloud 微服务大作业，主要围绕黑马头条实训项目展开，完成了微服务工程源码整理、数据库准备、部署说明编写以及提交材料整理。

## 2. 项目目标

项目目标是通过微服务架构方式完成内容平台的基础功能模块拆分，并具备本地运行与展示条件。

## 3. 技术选型

- Spring Boot
- Spring Cloud
- Spring Cloud Alibaba
- MySQL
- Nacos
- MinIO

## 4. 核心模块

- 用户服务
- 文章服务
- 自媒体服务
- 网关服务

## 5. 部署说明

部署时先导入数据库，再启动基础中间件，最后依次启动各微服务和前端页面。

## 6. 提交成果

最终已整理完成源码、数据库脚本、实训报告、答辩材料、分工表和 README 部署说明。

## 7. 后续优化

后续可进一步补充线上访问地址、更多业务功能和自动化测试。
